# Connection Bot – Digital Psychological Intervention System

## 📌 Overview
This project is a **Digital Psychological Intervention System** for college students.  
It provides:
- AI-guided first aid chatbot (Watson + console/WhatsApp/web)
- Supabase storage with offline JSON fallback
- Screening tools (PHQ-9, GAD-7, etc.)
- Mood/sleep tracker, music therapy, streaks
- Escalation logic and admin analytics
